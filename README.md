# TrustAI Admin Panel

## 1. Install dependencies
````bash
npm install
````

## 2. Run the development server
````bash
npm run dev
````


## 3. Buld The application
````bash
npm run build
````

## 🔍 4. Preview the production build locally
````bash
npm run preview
````
